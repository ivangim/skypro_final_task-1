Git

  • git init — превратить текущую папку в локальный репозиторий Git.
  • git status — проверить текущее состояние файлов (измененные, добавленные).
  • git add — добавить файл в индекс для подготовки к коммиту. Для всех файлов используйте git add .
  • git commit -m "Сообщение" — зафиксировать изменения в репозитории.
  • git push — отправить зафиксированные коммиты на удаленный сервер (например, GitHub).
